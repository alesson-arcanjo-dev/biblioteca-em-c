# 📚 Biblioteca em C

Sistema de biblioteca desenvolvido em linguagem C para cadastro e listagem de livros via terminal.

## 🚀 Funcionalidades
- Cadastro de livros (nome, autor, editora e edição)
- Listagem de livros cadastrados
- Menu interativo no terminal
- Controle de limite de registros

## 🛠️ Tecnologias utilizadas
- Linguagem C
- Compilador GCC

## ▶️ Como executar

```bash
gcc main.c -o biblioteca
./biblioteca
